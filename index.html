CSS properties

Property Value Description
font-family Georgia, Helvetica, "Times New Roman", sans-serif, serif, monospace defines the typeface of an element
font-size 16px, 72px, 2em controls the size of text displayed
font-weight normal, bold, bolder, lighter, 100, 900 defines how thin or thick text is displayed
font-style normal, italic text can appear in italics
text-align left, right, center set the text alignment of inline contents
line-height 1.5, 2 vertical spacing between lines of text
word-spacing 0.25em, 2px changes how far apart individual words are
letter-spacing 3px, 1em changes how far apart individual letters are
text-transform uppercase, lowercase makes all letters uppercase or lowercase
color red, #00FF36, rgb(76, 150, 71), hsl(180, 80%, 100%) defines foreground colour (colour of the text)
background-color black, #FC0, rgba(255, 0, 13, 0.3), hsla(240, 60%, 30%, 0.8) defines background colour (behind the text)
background-image url("picture.png") set the background of an element to an image
opacity 0, 1, 0.5 transparency of an element
box-sizing content-box, border-box controls the box model used by the browser
position static, relative, absolute, fixed, sticky specify the position of an element
z-index 1, 2, 3 how far back or forward an element appears on the page when it overlaps other elements
height 100px, 50% height of a content area
width 200px, 80% width of a content area
border 1px solid red, 3px dotted green, 10px dashed #56F896 shorthand property pro border-width, border-style, border-color
padding 10px, 20% space between the content area and the border
margin 0 auto, 20px spacing outside of an element’s border
overflow display, hidden, scroll dictates how HTML will render content that overflows its parent’s content area
visibility visible, hidden, collapse can hide or show elements
display inline, block, inline-block control how an element flows vertically and horizontally in a document
float left, right, none move elements as far left or as far right as possible
clear right, left, both, none clear an element’s left side, right side or both
