CSS properties

Text and colors

Property Value Description
font-family Georgia, Helvetica, "Times New Roman", sans-serif, serif, monospace defines the typeface of an element
font-size 16px, 72px, 2em controls the size of text displayed
font-weight normal, bold, bolder, lighter, 100, 900 defines how thin or thick text is displayed
font-style normal, italic text can appear in italics
text-align left, right, center set the text alignment of inline contents
line-height 1.5, 2 vertical spacing between lines of text
word-spacing 0.25em, 2px changes how far apart individual words are
letter-spacing 3px, 1em changes how far apart individual letters are
text-transform uppercase, lowercase makes all letters uppercase or lowercase
color red, #00FF36, rgb(76, 150, 71), hsl(180, 80%, 100%) defines foreground colour (colour of the text)
background-color black, #FC0, rgba(255, 0, 13, 0.3), hsla(240, 60%, 30%, 0.8) defines background colour (behind the text)
background-image url("picture.png") set the background of an element to an image
opacity 0, 1, 0.5 transparency of an element
position static, relative, absolute, fixed, sticky specify the position of an element
z-index 1, 2, 3 how far back or forward an element appears on the page when it overlaps other elements
overflow display, hidden, scroll dictates how HTML will render content that overflows its parent’s content area
visibility visible, hidden, collapse can hide or show elements
display inline, block, inline-block control how an element flows vertically and horizontally in a document
float left, right, none move elements as far left or as far right as possible
clear right, left, both, none clear an element’s left side, right side or both

Box model

Property Value Description
box-sizing content-box, border-box controls the box model used by the browser
height 100px, 50% height of a content area
width 200px, 80% width of a content area
border 1px solid red, 3px dotted green, 10px dashed #56F896 shorthand property pro border-width, border-style, border-color
padding 10px, 20% space between the content area and the border
margin 0 auto, 20px spacing outside of an element’s border

Flexbox

Property Value For Axis Description
display: flex changes an element to a block-level container with flex items inside of it
display: inline-flex allows multiple flex containers to appear inline with each other
justify-content flex-start, flex-end, center, space-around, space-between container main _ used to space items along the main (horizontal) axis
align-items flex-start, flex-end, center, baseline, stretch container cross | space items along the cross axis, for single row
align-content flex-start, flex-end, center, space-around, space-between, stretch container cross | is used to space rows along the cross axis, for multiple rows
flex-grow number (0,1,2,...), default value 0 items main _ specify if items should grow to fill a container and also which items should grow proportionally more or less than others
flex-shrink number (0,1,2,...), default value 1 items main _ specify which elements will shrink and in what proportions
flex-basis 50px, 100px, 200px,... items allows us to specify the width of an item before it stretches or shrinks
flex 1, 2, 100px; 2, 1, 200px;... items declare flex-grow, flex-shrink, and flex-basis all in one line
flex-wrap wrap, wrap-reverse, nowrap container main _ specifies that elements should shift along the cross axis if the flex container is not large enough
flex-direction row, row-reverse, column, column-reverse container ordering flex items vertically or horizontally
flex-flow column wrap container declare both the flex-direction and flex-wrap properties in one line

Grid

Property Value Description
display grid, inline-grid make the element CSS grid
grid-template-columns 100px 200px defines the number and sizes of the columns of the grid
grid-template-rows 50px 100px 50px defines the number and sizes of the rows of the grid
grid-template 200px 300px / 20% 10% 70% is a shorthand for defining both grid-template-rows and grid-template-columns in one line
repeat grid-template-columns:repeat(5, 1fr), grid-template-rows: repeat(10, 100px) five equal columns, ten rows with 100px height
minmax grid-template-columns: 100px minmax(100px, 500px) 100px; to prevent a row or column from getting too big or too small, second column in example 100-500px
row-gap 10px puts blank space between the rows of the grid
column-gap 20px puts blank space between the columns of the grid
gap 10px 20px shorthand for defining both row-gap and column-gap in one line
grid-row-start, grid-row-end grid-row-start: 1; grid-row-start: 3; makes elements span certain rows of the grid (example row 1 and 2), for grid item
grid-row 1 / 3, 1 / span 2 shorthand for grid-row-start and grid-row-end, for items
grid-column-start, grid-column-end grid-column-start: 1; grid-column-end: span 3; makes elements span certain columns of the grid, for items
grid-column 1 / span 2 shorthand for grid-column-start and grid-column-end, for items
grid-area 5 / 1 / span 2 / span 2 shorthand for grid-row-start, grid-column-start, grid-row-end, and grid-column-end, all in one line, for items
